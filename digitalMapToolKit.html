Some useful tools for making maps

Mostly free and freemium tools (with some ESRI exceptions) for making maps because thats all this contributor can afford.
Get started using the following tools to make maps!

Web Maps

Leaflet

Leaflet.js is a JavaScript library primarily for making web maps. You can display a raster or vector tiles and if you have a geoJSON data set you can layer those guys on your tiles.

Mapbox

Mapbox.js allows you to do a bunch of jam that's a bit more customizable than leaflet, don't think it's as lightweight though.

Open Street Map

Add data to Open Steet Map!

CartoDB

Need a simple place to deal with your geodata, or attach non geo data to geo data, CartoDB is perfect for you. Plus there're some really straight forward wizards for making that data into geo visualizations.

Google Maps

Eh google maps, they explain a lot at google, use if you're boring.
